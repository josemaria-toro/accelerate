# Description
Accelerate it's a framework based on .NET, to help us to improve the time to market, providing libraries with the most common components for our applications. 
# Libraries
Accelerate is composed for several libraries in order to help us in all layers of our applications:
- Libraries with common objects for application layer
- Libraries for managing data and objects in cache
- Libraries to make easier the data access
- Libraries with common objects for domain layer
- Library with the exceptions catalog
- Libraries to perform logging in some platforms and systems
- Libraries to perform telemetry in some platforms and systems
- Libraries to help us to implement our tests classes
- Libraries with common objects for web layer
